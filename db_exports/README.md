# Importer les collections MongoDB

Ce dossier contient les exports des collections MongoDB au format JSON.

## Importer avec MongoDB Compass
1. Ouvrez MongoDB Compass et connectez-vous à votre base de données.
2. Cliquez sur la base de données cible, puis sur la collection à importer (ou créez-la si elle n'existe pas).
3. Cliquez sur "Add Data" > "Import JSON".
4. Sélectionnez le fichier .json correspondant (ex : projects.json).
5. Validez l'import.
6. Répétez pour chaque collection.
